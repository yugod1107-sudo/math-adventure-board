# 數學探險棋

這是第一版單機筆電原型。用瀏覽器打開 `index.html` 就可以玩。

## 目前功能

- 1 到 4 位玩家
- 玩家可選企鵝、浣熊、花鹿、海豚
- 30 格棋盤
- 棋盤從左上走到右下
- 遊戲畫面在筆電橫向視窗中盡量維持單一畫面
- 同一格最多四位玩家都會顯示
- 擲骰子有骰子臉點數動畫
- 棋子移動會逐格跳動，擲骰與特殊格說明會停留較久
- 回合資訊精簡顯示在上方狀態列
- 右上角角色使用 `assets/ntuk charaters.png` 裁切出的探險隊圖片
- 100 以內加減法
- 九九乘法範圍
- 整除除法
- 可選難度：小一 20 內加減、小二 100 內加減、小三加減乘除
- 四選一答題
- 答對後擲骰前進
- 答錯原地不動並換下一位
- 星星格、泥巴格、禮物格

## 試玩方式

方式一：直接用瀏覽器開啟：

```text
math-adventure-board/index.html
```

方式二：用本機伺服器開啟：

```text
雙擊 start-game.bat
瀏覽器打開 http://127.0.0.1:5173
```

之後如果要用 iPad 測試，可以在筆電開本機伺服器，再讓 iPad 用同一個 Wi-Fi 連進來。
